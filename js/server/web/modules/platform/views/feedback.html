  • 为什么微云上传文件不能超过1G？

    您好，由于业务功能调整，上传单个文件/视频，普通用户单日上传上限不能超过1G，会员用户单日上传上限不能超过20G，有特殊需求的用户可以开通会员提升单日上传上限。

    微云无法上传提示错误代码510070？

    您好，如果您在文件上传的时候遇到相关的提示，建议您对您当前的浏览器使用修复工具修复后重新上传 。

    微云无法上传提示错误代码510065？

    您好，如果您在文件上传的时候遇到相关的提示，建议您对您当前的浏览器使用修复工具修复后重新上传。

  • 客户端首次登陆后无法下载，提示目录加载中？

    您好，由于首次登陆后目录需要重新拉取数据，建议您等目录加载完毕后再下载文件。

  • 账号系统大升级，微信与QQ数据打通合并

    1、数据互通
    为进一步提升用户微云使用体验，微云项目组决定升级存储数据，打通合并微信账号与QQ账号数据。

    2、账号融合
    合并后的数据可以使用QQ或微信账号两者任一账号登录。

    3、微信绑定
    在微信App内绑定QQ账号的用户，微信号与QQ号数据将会打通合并，如未绑定微信QQ的用户需要帐号合并，则需在【微信】App内绑定QQ账号。

    4、储存容量
    合并后网盘容量自动升级为QQ号或微信号二者较大的容量。

    5、合并时间
    2015年12月14日至2015年12月24日。
    PS：由于融合工程浩大，时间会根据进度顺延，请广大用户理解及见谅

    怎么将不用的文件删除/找回？

    您可以通过微云目录->选中您不需要的文件->点击垃圾桶图标->选择确定进行删除，如果您想找到删除的文件，您可以通过微云工具->回收站进行还原即可（删除的文件如超过30天将无法进行还原）

    怎么关闭独立密码？

    请您使用手机登陆微云=》工具=》设置=》安全与隐私=》独立密码=》关闭独立密码=》输入当前设置的独立密码后即可关闭完成

    如何分享微云网盘上的文件给好友？

    1、如果您想把某一个文件分享给好友，可以先进入微云网盘=》点击页面下方的“分享”=》选择要分享的文件。
    2、微云网盘会自动替该文件生成一个网页链接，您可以选择用邮件、短信或者其他手机应用（比如微博、微信和QQ等)把网页链接发送给好友，而您的好友可以直接使用电脑或手机的浏览器打开链接即可下载该文件。
    温馨提示：只要分享链接的文件未删除，分享链接将会一直存在。

    积分兑换规则

    微云积分兑换规则
    Q：多少积分才能兑换一个月微云会员？
    A：1000积分兑换一个月微云会员。

    Q：积分兑换只能兑换一次还是无限次？
    A：只要积分足够，可以无限次数兑换。

    Q：积分不够怎么办？
    A：积分不够的情况下，是不能兑换的，需要等获取足够的积分才能兑换，坚持签到即可积累更多积分。

    Q：兑换成功后，在哪里查看是否已开通微云会员？
    A：兑换成功后，在微云APP“工具”页面顶部可见微云会员图标已点亮。

    Q：兑换成功后，在哪里查看微云会员的有效期？
    A：兑换成功后，一般当日即可开通微云会员，在微云会员介绍页面可见会员到期日期。

    Q：已经是微云会员的用户，又兑换了微云会员，在哪里查看？
    A：在微云会员介绍页面，可见微云会员到期日期延长了31天。
