微云2.0对产品功能进行了全方位更新，将由单一的存储服务向全能型的聚合服务产品进化。无论是产品功能还是产品定位，微云都在延续云存储产品核心价值的基础上，朝着全能化的方向发展。微云2.0除了可以依托10TB高速云存储空间向用户提供照片、文件的存储与跨平台传输功能外，进一步拓展了云存储产品的可能性，将QQ影音内核、应用宝、写笔记、二维码扫描等功能聚合进微云。

依托于云技术向用户提供全方位的服务，将会是个人云存储产品新的发展趋势，微云2.0也将以全面变革的姿态，通过当下云存储产品所不具备的强大功能服务体系，将网络存储市场推向一个崭新的全能服务时代。
